    • 大三信安专业在读，目前在知道创宇实习。
    • 研究方向：windows及Linux系统内核，网络与信息安全攻防、病毒木马分析及开发、逆向破解、渗透测试、全系列安全产品及服务、软件开发（汇编、C、C++、Go、Python、Shell等）、软件测评。

    • 熟练的 Linux 服务器开发技能， 熟练掌握 C/C++编程；
    • 熟悉计算机原理、 算法与数据结构、 HTTP、 TCP/IP 协议；
    • 具备常用 Qt 界面开发的基本技巧和常用技能， 能够实现简单的前台界面开发；
    • 掌握常用数据库(Oracle、 MySQL、 MongoDB、 Redis)的操作技巧和简单开发；
    • 掌握跨平台兼容开发与调试、 Git 代码管理；
    • 掌握高性能服务器解决方案(负载均衡)；
    • 掌握分布式存储解决方案(redis 集群/mysql 集群/ mongodb 集群)；
    • 掌握项目压力测试和单元测试， 掌握服务器逻辑开发；
    • 具备独立设计和研发游戏业务逻辑能力。

    # Go方面

    • 具备 go 语言高并发后台开发能力；
    • 具备 docker 虚拟化容器技术；
    • 具备分布式微服务架构设计能力；
    • 解决企业容器虚拟化技术；
    • 解决企业从资源的利用率和可用性优化方案；
    • 能解决传统 PaaS 企业， 采用 Micro Service(微服务)的方式来实现他们的应用。

    # Linux 运维方面

    • 大一时参加过红帽 RHCSA、 RHCE 考试认证。
    • 熟悉 Unix/Linux 下 Shell 编程开发；
    • 了解 RAID 与 LVM 磁盘阵列技术；
    • 熟悉 iptables 与 Firewalld 防火墙；
    • 熟练使用 ssh、 Apache、 Vsftpd、 Samba/NFS、 Bind、 DHCP、 Postfix 与 Dovecot、 Squid、 iSCSI、 MariaDB、PXE+Kickstart、 LNMP 等服务及其相关部署 。

    # 信息安全方面

    • 具备市场主流网络安全攻防技术；
    • OWASP 中国区会员；
    • 信息安全科班出生、 长期活跃在 FreeBuf、 看雪安全论坛；
    • 熟悉各种渗透测试工具并且对其原理有一定了解（不仅限于 Burpsuite、 sqlmap、 appscan、 AWVS、nmap、 MSF， cobalt strike 等） ；
    • 熟悉全系列安全产品及服务；
    • 熟练使用 DTdebug、 OnlyDebug、 windbg、 IDA 等调试、 逆向工具；
    • 熟悉 PE 文件格式；
    • 熟悉 Win32 反汇编；
    • 具备流量监测等核心安全研发能力；
    • 解决国内外敌对势力通过隐匿通道(如 VPN、 即时通信软件)有限监控手段现状问题；
    • 解决私有加密应用和网络异常通信行为， 并能对加密流量进行还原和审计。

    # 网络方面

    • 熟悉 CISCO 设备管理与配置；
    • 熟悉 RIP、 EIGRP、 OSPF、 BGP 路由交换协议；
    • 熟悉 VLAN、 STP、 VTP、 链路聚合、 vlan 间路由、 访问控制列表(ACL)；
    • 熟悉 DHCP、 WLAN、 DNS、 NTP、 Syslog 与 SNMP；
    • 熟悉广域网技术、 NAT 技术、 IPv6 技术、 GRE 技术；
    • 熟悉 IPsecVPN 技术、 AAA 与 802.1X、 网络防火墙、 网关冗余协议 HSRP 与 VRRP。
